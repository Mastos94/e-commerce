# Synergy - E-Commerce Web Application

## Laboratory Work #1-4: Collaborative Web Development

### Team Project
An e-commerce web application developed collaboratively using modern software development practices.

### Technology Stack
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **Frontend**: HTML/CSS/JavaScript (Client-side MVC)
- **Version Control**: Git + GitHub

### Team Roles (2-3 members)
- **Backend Developer**: Server-side logic, database, API endpoints
- **Frontend Developer**: Client-side interface, user experience, async operations
- **Full-Stack/DevOps** (if 3 members): Testing, security, deployment, code quality

### Architecture
- MVC (Model-View-Controller) pattern
- Repository pattern for data access
- Async/await for all I/O operations
- JSON for data exchange

### Features
- Product catalog with search and filtering
- User registration and authentication
- Shopping cart management
- Order processing
- User profile and order history
